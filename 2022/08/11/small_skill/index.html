秒数转换为 天 小时 分钟 秒 的格式

  • num为需要转换的秒数

    //天数计算
int days = (num)/(24*3600);
  
//小时计算
int hours = (num)%(24*3600)/3600;
  
//分钟计算
int minutes = (num)%3600/60;
  
//秒计算
int second = (num)%60;
